About the Project

Automating basketball analytics to provide clarity, not clutter.

Why I Built This

In the world of sports analytics, there is often too much noise. I created this platform to bridge the gap between complex web scraping and clean data visualization.

This isn't just a stats table; it's an automated pipeline that ensures every piece of data shown—from Power Ratings to Defensive metrics—is processed, validated, and ready for scouting.

Python Django Pandas BeautifulSoup

Daily Scraping

Automated scripts fetch raw HTML tables from third-party sports providers every 24 hours.

Pandas Filtering

Raw data is cleaned and filtered to retain only the high-impact metrics requested for scouting.

Relational Mapping

Cleaned stats are stored in a structured Django database for fast, team-centric navigation.
